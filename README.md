# HTML常用标签
* 超链接a标签
```
herf取值:
1.网址
2.路径
3.伪协议
4.id
```
```
targrt取值:
指定在哪个文件打开
1._blank
2._top
3._parent
4._self
```
* 图片img标签
```
1.作用:发出请求，展示图片
2.属性：src/alt/width/height
```
* 表格table标签
```
相关标签
1.thead
2.tbody
3.tfoot
4.tr(行)
5.th(表头)
6.td(数据)
```
```
相关样式
1.table_layout:auto
2.border_collapse:collapse(是否合并)
3.border_spacing:0(是否有间距)
```
